## 📋 Sobre o Projeto

Mar Quente HUB é um sistema completo de gestão empresarial desenvolvido com React e Supabase.

### ✨ Funcionalidades

- 👥 Gestão de Usuários (Admin, Manager, Membro)
- 📊 Dashboard Interativo
- 📁 Gestão de Projetos
- ✅ Sistema de Tarefas
- 🏢 Departamentos
- 🎫 Sistema de Tickets
- 📅 Calendário de Lançamentos

## 🚀 Tecnologias

- React 18.3
- Vite
- Supabase
- TailwindCSS
- Shadcn/ui
